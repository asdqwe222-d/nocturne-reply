Troubleshooting Tips

  1. Server not running: Make sure the server is started by running cd gpt-relay-server && node server.js in a terminal.
  2. CORS issues: When accessing from a file:// URL, browsers enforce security restrictions. Try one of these solutions:
    • Open the test-prompt.html file in a web server instead of directly from file system
    • Update your server's CORS settings in the .env file to allow all origins
    • Click the "Open Test Page in Browser" button above (once server is running)
  3. Port conflict: Something else might be using port 3000. Check if the server started without errors.
  4. Firewall blocking: Your firewall might be blocking connections to localhost:3000.
